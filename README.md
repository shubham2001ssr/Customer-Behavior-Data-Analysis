# Customer Behavior Data Analyst Portfolio Project
**Author:** Shubham ([@shubham2001ssr](https://github.com/shubham2001ssr))

## 📌 Project Overview
This project represents a complete, industry-standard, end-to-end data analytics workflow. It is designed to mirror the real-world responsibilities of professional analysts in modern business environments by encompassing all critical stages of data analysis: from data preparation and modeling to insight generation, visualization, and reporting.

The goal of this project is to simulate a corporate-grade data analytics workflow, demonstrating the ability to translate raw retail customer data into strategic business intelligence through:

- **Data Preparation, Modeling & Exploratory Data Analysis (Python):** Cleaning and transforming the raw dataset for analysis using Python libraries (Pandas, NumPy).
- **Data Analysis (SQL):** Simulating business transactions and running queries to extract insights on customer segments, loyalty, and purchase drivers.
- **Visualization & Insights (Power BI):** Building an interactive dashboard that highlights key patterns and trends, enabling stakeholders to make data-driven decisions.
- **Reporting & Presentation:** Summarizing key findings and actionable business recommendations in a clear project report.

## 🛠️ Tech Stack
- **Python** (Pandas, NumPy, Matplotlib/Seaborn for EDA)
- **SQL** (MySQL / PostgreSQL / MS SQL Server)
- **Power BI**
- **Jupyter Notebook**

## 📂 Project Structure
- `Customer_Shopping_Behavior_Analysis.ipynb`: Python notebook for data import, exploration, cleaning, and pushing data to the SQL database.
- `customer_behavior_sql_queries.sql`: SQL scripts used to answer key business questions and analyze customer trends.
- `customer_behavior_dashboard.pbix`: Power BI file containing the interactive reporting dashboard.
- `reports/`: Folder containing the final project report and presentation deck summarizing insights.

## 🚀 How to Run and Use This Project

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shubham2001ssr/customer-trends-data-analysis.git
   cd customer-trends-data-analysis
   ```

2. **Run the Python Setup & Data Cleaning:**
   - Open `Customer_Shopping_Behavior_Analysis.ipynb`.
   - Run the cells to perform Exploratory Data Analysis (EDA) and data cleaning.
   - Configure your SQL database credentials in the notebook to load the cleaned data into your local SQL server.

3. **Perform SQL Analysis:**
   - Open `customer_behavior_sql_queries.sql` in your preferred SQL client (e.g., pgAdmin, MySQL Workbench).
   - Execute the queries to uncover insights into customer segments, sales trends, and behavioral patterns.

4. **Explore the Power BI Dashboard:**
   - Connect your SQL database to Power BI.
   - Open `customer_behavior_dashboard.pbix` to interact with the visualizations and view the final dashboard.
